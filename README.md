# Employee Management System (EMS)

A simple Employee Management System built using React.  
This project allows users to add, edit, delete, and view employee records through a clean user interface.

## 🚀 Features

- Add employees
- Edit employee details
- Delete employees
- View employee list
- Responsive design

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- CSS / Tailwind CSS

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/AbhishekRangra98/EMS.git

2. Go to the project folder
cd EMS

3. Install dependencies
npm install

4. Run the project
npm run dev
